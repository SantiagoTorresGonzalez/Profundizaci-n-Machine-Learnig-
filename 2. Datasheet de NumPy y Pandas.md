B) NumPy – Operaciones estadísticas y funciones avanzadas

- **np.mean():** La funcion del comando es la de calcular el promedio de los elementos dentro de un array sumando todos los valores y el resultado lo divide en el total de elementos del array
  
- **np.median():** El comando calcula la mediana de un array cuando los datos están ordenados; existen dos casos, el primer caso es cuando la cantidad de datos dentro del array es impar devuelve el valor del medio pero si la cantidad de datos es par devuelve el promedio de los dos valores centrales
  
- **np.std():** Este comando entrega un valor el cual inidca que tan diferentes o que tan dispersos estan los números tomando como referencia el promedio, del dato que se entrega se puede decir que si este valor es bajo entonces significa que hay poca variacion y si es alto significa exite una mayor dispersión
  
- **np.sum():** La función del comando es entregar la suma de todos los elementos que esten dentro de un array y lo devuelve como un unico valor
  
- **np.cumsum():** Esta funcion suma los valores de un array, pero al contrario del comando (np.sum) que entrega un solo valor este suma los valores del array de manera progresiva y forma un nuevo array con los resultados
  
- **np.arange():** Esta funcion entrega un array con una secuencia de números definido por un valor inicial, un valor final (El cual no se incluye en el resultado final) y un tamaño de paso que indica cada cuánto se incrementan los números
  
- **np.linspace():** La función del comando es generar un array de números igualmente espaciados entre un valor inicial y un valor final, tambien se puede definir la cantidad de valores que se quieren tener en el array
  
- **np.random.rand():** La función del comando es generar números aleatorios distribuidos de forma pareja entre 0 a 1 indicando la cantidad de valores generados dentro del array
  
- **np.random.randn():** La función del comando es generar números aleatorios que siguen una distribución normal con media igual a 0 y desviación estándar igual a 1
  
- **np.dot():** La función del comando es realizar el producto punto entre vectores o la multiplicación entre matrices; para vectores el comando devuelve un número escalar mientras que para matrices devuelve otra matriz
  
- **np.linalg.inv():** La función del comando es calcular la matriz inversa de una matriz cuadrada. La inversa es aquella matriz que, al multiplicarse por la matriz original, da como resultado la matriz identidad
